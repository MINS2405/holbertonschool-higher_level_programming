# Python - if/else, loops, functions

This project is part of the Python - if/else, loops, functions course at Holberton School. It covers various aspects of control flow in Python, including if/else statements, loops (for and while), functions, and basic programming concepts.

## Objectives

- Understand why indentation is important in Python.
- Learn to use if and if/else statements.
- Understand how to use comments.
- Learn to assign values to variables.
- Use while and for loops effectively.
- Understand the break and continue statements.
- Use else clauses with loops.
- Know what the pass statement does and when to use it.
- Learn to use the range function.
- Understand what a function is and how to use it.
- Understand the scope of variables.
- Learn about tracebacks and error handling.
- Understand arithmetic operators and how to use them.

## Tasks

1. **Positive anything is better than negative nothing** : Write a program that prints whether a randomly generated number is positive, negative, or zero.
2. **The last digit** : Print the last digit of a randomly generated number.
3. **I sometimes suffer from insomnia. And when I can't fall asleep, I play what I call the alphabet game** : Print the ASCII alphabet in lowercase without a new line.
4. **When I was having that alphabet soup, I never thought that it would pay off** : Print the ASCII alphabet except for 'q' and 'e'.
5. **Hexadecimal printing** : Print numbers from 0 to 98 in decimal and hexadecimal.
6. **00...99** : Print numbers from 0 to 99 with two digits each.
7. **Inventing is a combination of brains and materials. The more brains you use, the less material you need** : Print all possible combinations of two different digits.
8. **islower** : Write a function that checks if a character is lowercase.
9. **To uppercase** : Write a function that prints a string in uppercase.
10. **There are only 3 colors, 10 digits, and 7 notes; it's what we do with them that's important** : Write a function that prints the last digit of a number.
11. **a + b** : Write a function that adds two integers.
12. **a ^ b** : Write a function that computes a to the power of b.
13. **Fizz Buzz** : Write a function that prints numbers from 1 to 100, replacing multiples of three with "Fizz", multiples of five with "Buzz", and multiples of both with "FizzBuzz".

## Execution

To run the files, ensure you have Python 3 installed on your system. Use the command `python3` to execute each Python file.
