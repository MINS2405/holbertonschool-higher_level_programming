RESTful API Project
This project explores the fundamental concepts of RESTful APIs, covering both consumption and development of APIs using Python and command-line tools.

Learning Objectives
Understand the basics of HTTP/HTTPS

Consume APIs using command-line tools (curl)

Use Python to interact with APIs

Develop a simple API using Python's http.server module

Create a more robust API with Flask

Implement security and authentication for APIs

Document APIs according to OpenAPI standards

Project Structure
The project is divided into several tasks, each focusing on a specific aspect of RESTful API development and usage:

HTTP/HTTPS Basics: Understanding the differences between HTTP and HTTPS, HTTP request/response structure, and common status codes.

API Consumption with curl: Using curl to interact with APIs from the command line.

API Consumption with Python: Using the requests library to make API calls and process JSON data.

API Development with http.server: Creating a simple API using Python's http.server module.

API Development with Flask: Building a more robust API using the Flask framework.

API Security & Authentication: Implementing security mechanisms to protect APIs.

API Documentation with OpenAPI: Learning to document APIs in a standardized way.

Prerequisites
Python 3.x

pip (Python package manager)

curl (for command-line tests)

Installation
Clone this repository:

text
git clone https://github.com/your-username/restful-api-project.git
Install dependencies:

text
pip install -r requirements.txt
Usage
Each task is contained in its own Python file. To run a specific task, use the following command:

text
python filename.py
For example, to run task 2:

text
python task_02_requests.py
