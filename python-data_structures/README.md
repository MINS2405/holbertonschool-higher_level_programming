# Python - Data Structures: Lists, Tuples

This project is part of the Python - Data Structures: Lists, Tuples course at Holberton School. It covers various aspects of working with lists and tuples in Python, including their differences, common methods, and how to use them effectively.

## Objectives

- Understand what lists and tuples are and how to use them.
- Learn the differences and similarities between strings and lists.
- Understand the most common methods of lists and how to use them.
- Learn how to use lists as stacks and queues.
- Understand what list comprehensions are and how to use them.
- Learn about tuples and when to use them instead of lists.
- Understand what sequences are and how to use tuple packing and sequence unpacking.
- Learn about the del statement and how to use it.

## Tasks

1. **Print a list of integers** : Write a function that prints all integers of a list.
2. **Secure access to an element in a list** : Write a function that retrieves an element from a list without raising an error if the index is out of range.
3. **Replace element** : Write a function that replaces an element in a list at a specific position.
4. **Print a list of integers in reverse** : Write a function that prints all integers of a list in reverse order.
5. **Replace in a copy** : Write a function that replaces an element in a copy of a list without modifying the original list.
6. **Can you C me now?** : Write a function that removes all characters 'c' and 'C' from a string.
7. **Lists of lists = Matrix** : Write a function that prints a matrix of integers.
8. **Tuples addition** : Write a function that adds two tuples element-wise.
9. **More returns!** : Write a function that returns a tuple with the length of a string and its first character.
10. **Find the max** : Write a function that finds the biggest integer in a list.
11. **Only by 2** : Write a function that returns a list of boolean values indicating whether each integer in the input list is divisible by 2.
12. **Delete at** : Write a function that deletes the item at a specific position in a list.
13. **Switch** : Complete the source code to switch the values of two variables.

## Execution

To run the files, ensure you have Python 3 installed on your system. Use the command `python3` to execute each Python file.
