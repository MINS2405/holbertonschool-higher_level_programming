# Python - Hello, World

This project is part of the Python - Hello, World course at Holberton School. It covers basic concepts of Python programming, including using the Python interpreter, printing text and variables, working with strings, and understanding indexing and slicing.

## Objectives

- Learn to use the Python interpreter.
- Understand how to print text and variables using `print`.
- Learn to use strings and perform basic operations on them.
- Understand indexing and slicing in Python.
- Familiarize yourself with the official Python coding style and how to check your code with pycodestyle.

## Tasks

1. **Hello, print** : Write a Python script that prints exactly "Programming is like building a multilingual puzzle" followed by a new line.
2. **Print integer** : Complete the source code to print an integer stored in a variable, followed by "Battery street", followed by a new line.
3. **Print float** : Complete the source code to print a float with a precision of 2 digits.
4. **Print string** : Complete the source code to print a string three times, followed by its first 9 characters.
5. **Play with strings** : Complete the source code to print "Welcome to Holberton School!".
6. **Copy - Cut - Paste** : Complete the source code to extract parts of a string.
7. **Create a new sentence** : Complete the source code to print "object-oriented programming with Python".
8. **Easter Egg** : Write a Python script that prints "The Zen of Python" by Tim Peters.

## Execution

To run the files, ensure you have Python 3 installed on your system. Use the command `python3` to execute each Python file.
