Python Learning Project

This project introduces fundamental Python concepts and best practices for beginners.


Objectives:

Learn Python basics: interpreter, syntax, and data types

Understand control flow: loops, conditionals, and functions

Explore modules and command-line arguments


Apply PEP 8 style guidelines using pycodestyle

Key Topics:

String formatting and manipulation

Loops and conditional statements

Functions and variable scope

Importing and creating modules

Command-line argument handling


Requirements:

Python 3.8+ (Ubuntu 20.04 LTS)

pycodestyle 2.7.*

All scripts must be executable and start with #!/usr/bin/python3
