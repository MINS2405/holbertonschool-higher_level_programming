# Python - Classes and Rectangles

This project is part of the Python - Classes and Rectangles course at Holberton School. It covers various aspects of object-oriented programming (OOP) in Python, including defining classes, attributes, and methods, using rectangles as an example.

## Objectives

- Understand and use classes and instances in Python.
- Define private and public attributes within a class.
- Use special methods like `__init__`, `__str__`, and `__repr__`.
- Understand the difference between class and instance attributes.
- Learn to use class methods and static methods.
- Implement rectangles with properties like width and height.
- Calculate the area and perimeter of a rectangle.
- Display a visual representation of a rectangle.

## Tasks

1. **Simple Rectangle** : Define an empty `Rectangle` class.
2. **Real Definition of a Rectangle** : Add private instance attributes for width and height with properties to access and modify them.
3. **Area and Perimeter** : Add methods to calculate the area and perimeter.
4. **String Representation** : Implement `__str__` and `__repr__` methods to display the rectangle.
5. **Detect Instance Deletion** : Print a message when an instance is deleted.
6. **Count Instances** : Use a class attribute to count the number of instances.
7. **Change Representation** : Allow changing the symbol used for string representation.
8. **Compare Rectangles** : Add a static method to compare two rectangles.
9. **A Square is a Rectangle** : Add a class method to create a square.

## Execution

To run the files, ensure you have Python 3 installed on your system. Use the command `python3` to execute each Python file.
