# Python - More Data Structures

This project is part of the Python - More Data Structures course at Holberton School. It covers various aspects of data structures in Python, including sets, dictionaries, lambda functions, and the map, filter, and reduce functions.

## Objectives

- Understand and use sets and their common methods.
- Understand and use dictionaries and their methods.
- Learn to use lambda functions and the map, filter, and reduce functions.
- Know when to use sets versus lists.
- Know how to iterate over a set.
- Understand what dictionaries are and how to use them.
- Know when to use dictionaries versus lists or sets.
- Understand what a key is in a dictionary.
- Know how to iterate over a dictionary.

## Tasks

1. **Squared simple** : Write a function that computes the square value of all integers in a matrix.
2. **Search and replace** : Write a function that replaces all occurrences of an element with another in a new list.
3. **Unique addition** : Write a function that adds all unique integers in a list.
4. **Present in both** : Write a function that returns a set of common elements in two sets.
5. **Only differents** : Write a function that returns a set of elements present in only one set.
6. **Number of keys** : Write a function that returns the number of keys in a dictionary.
7. **Print sorted dictionary** : Write a function that prints a dictionary by ordered keys.
8. **Update dictionary** : Write a function that updates or adds a key/value pair in a dictionary.
9. **Simple delete by key** : Write a function that deletes a key in a dictionary.
10. **Multiply by 2** : Write a function that returns a new dictionary with all values multiplied by 2.
11. **Best score** : Write a function that returns a key with the biggest integer value.
12. **Multiply by using map** : Write a function that returns a list with all values multiplied by a number using map.
13. **Roman to Integer** : Write a function that converts a Roman numeral to an integer.

## Execution

To run the files, ensure you have Python 3 installed on your system. Use the command `python3` to execute each Python file.
