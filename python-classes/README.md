# Python - Classes and Objects

This project is part of the Python - Classes and Objects course at Holberton School. It covers various aspects of object-oriented programming (OOP) in Python, including defining classes, attributes, methods, and understanding the differences between classes and objects.

## Objectives

- Understand what OOP is and its benefits.
- Learn about "first-class everything" in Python.
- Understand what a class and an object are.
- Learn about attributes (public, protected, private) and how to use them.
- Understand the role of the `self` parameter.
- Learn about methods, including the special `__init__` method.
- Understand data abstraction, encapsulation, and information hiding.
- Learn about properties and how they differ from attributes.
- Understand how to dynamically create attributes for instances.
- Learn how Python finds attributes in objects and classes.

## Tasks

1. **My first square** : Define an empty class `Square`.
2. **Square with size** : Add a private instance attribute `size` to the `Square` class.
3. **Size validation** : Implement validation for the `size` attribute to ensure it is a non-negative integer.
4. **Area of a square** : Add a method to calculate the area of the square.
5. **Access and update private attribute** : Implement getter and setter methods for the `size` attribute.
6. **Printing a square** : Add a method to print a visual representation of the square using `#`.
7. **Coordinates of a square** : Add a private instance attribute `position` to represent the coordinates of the square.

## Execution

To run the files, ensure you have Python 3 installed on your system. Use the command `python3` to execute each Python file.
