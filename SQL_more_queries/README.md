# SQL - More Queries

This project is part of the SQL - More Queries course at Holberton School. It covers advanced SQL concepts, including user management, database design, and complex queries.

## Objectives

- Learn to create new MySQL users and manage their privileges.
- Understand how to use PRIMARY and FOREIGN KEY constraints.
- Learn to use NOT NULL and UNIQUE constraints.
- Understand how to retrieve data from multiple tables using JOINs.
- Learn about subqueries and how to use them.
- Understand the difference between INNER JOIN, LEFT JOIN, RIGHT JOIN, and FULL OUTER JOIN.
- Learn to use UNION and MINUS operators to combine query results.
- Understand how to design databases using UML and SQL.

## Tasks

1. **My Privileges!** : List all privileges of the MySQL users user_0d_1 and user_0d_2.
2. **Create User** : Create a new MySQL user with all privileges.
3. **Read User** : Create a new MySQL user with only SELECT privileges on a specific database.
4. **Always a Name** : Create a table with a NOT NULL constraint on a column.
5. **ID Can't Be Null** : Create a table with an ID column that has a default value.
6. **Unique ID** : Create a table with a UNIQUE constraint on the ID column.
7. **States Table** : Create a table for states with an auto-incrementing ID.
8. **Cities Table** : Create a table for cities with a FOREIGN KEY referencing the states table.
9. **Cities of California** : List all cities in California without using the JOIN keyword.

## Execution

To run the SQL scripts, ensure you have MySQL 8.0 installed on your system. Use the command `mysql` to execute each SQL file.
